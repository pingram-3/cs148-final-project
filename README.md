## Installation

Clone this package into your `catkin_ws/src`:

```bash
cd ~/catkin_ws/src
git clone https://github.com/yourname/cs148_final_project.git
cd ..
catkin_make
source devel/setup.bash
```

## Example Usage
```bash
rosrun cs148_final_project occupancy_grid.py
rosrun cs148_final_project dijkstra_planner.py
rosrun cs148_final_project path_follower.py
```
